# ミニモンクエスト

ファイアーレッド風の操作感を目指した、完全オリジナルの自作モンスターRPG(Webアプリ/PWA)。
任天堂の素材・名称は一切使っていないので、配布・公開も自由です。

**▶ 遊ぶ: https://yudaiakutsu.github.io/minimon-quest/**
(iPhoneはSafariで開いて 共有 →「ホーム画面に追加」でアプリ化)

## 遊び方(Mac上で)

```sh
cd ~/minimon-quest
python3 -m http.server 8123
```

ブラウザで http://localhost:8123 を開く。

- 移動: 矢印キー / 画面の十字キー
- A(決定): Z キー / 画面の A ボタン
- B(キャンセル): X キー / 画面の B ボタン
- メニュー: Enter / START ボタン

## iPhone で遊ぶ

1. Mac で上記サーバーを起動したままにする
2. iPhone を **同じ Wi-Fi** につなぐ
3. Safari で `http://<MacのIPアドレス>:8123` を開く
   (IPアドレスは `ipconfig getifaddr en0` で確認)
4. 共有ボタン →「ホーム画面に追加」でアプリのように起動できる
   (一度読み込めばオフラインでも動作)

## いまある内容(ver 1.0)

- ハジマリタウン(自宅・研究所)+ 1ばんルート
- 御三家: ヒバナル(炎)/ シズクン(水)/ ハッパミ(草)
- 野生ミニモン: コトリス・ムシマル・デンリス
- ターン制バトル(タイプ相性・急所・能力変化・すばやさ順)
- モンカプセルでの捕獲 / 経験値・レベルアップ・わざ習得
- ライバル「シン」戦(こちらに不利なタイプを選んでくる)
- ママに話しかけると全回復 / レポート(セーブ)

## ファイル構成

- `js/data.js` … モンスター・わざ・タイプ相性・マップ・ドット絵の定義
- `js/game.js` … エンジン本体(描画・移動・会話・バトル・セーブ)
- `index.html` / `style.css` … 画面とタッチコントローラー
- `manifest.webmanifest` / `sw.js` / `icon-*.png` … PWA(ホーム画面アプリ化・オフライン対応)

## 拡張のしかた

- **ミニモン追加**: `data.js` の `SPECIES` と `PIXEL`(16x16ドット絵)に追記
- **わざ追加**: `data.js` の `MOVES` に追記
- **マップ追加**: `data.js` の `MAPS` に文字グリッドで追記し、`warps` / `edges` でつなぐ
- **NPC・イベント追加**: 各マップの `npcs` に `talk` 関数を書く

ゲームを更新したら `sw.js` の `CACHE` のバージョン番号を上げると確実に反映されます
(現在はネットワーク優先なので通常は自動反映)。
